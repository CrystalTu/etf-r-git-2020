1 Class Introduction

Topics

  1. R Project Set Up
  2. git and hosting a git repository
  3. Introductory git commands
  4. Sharing a project via git
  5. Bonus Differential Equations in R

1.1 Expectations

By attending this tutorial, we assume you have…

  1. working knowledge of R and RStudio
  2. conteporary tidy coding concepts
  3. a need to work collaborativly on a project
  4. a healthy appreciation of Dad Jokes

Give it a shot, if its too much or too little, will not be offended if you leave.

Its okay to struggle.

1.2 Prerequisites

  1. R 3.6.x or newer
  2. RStudio 1.2.x or newer
  3. Git
  4. Git Bash (included with Git for Windows) or another command-line application of your choice capable of using Git

1.3 Distance Learning Challenges

  1. Don’t be afraid to ask questions - both verbally and in chat
  2. If you miss something we said, its likely others have too
  3. Its difficult the know the speed of the class so please communicate!

1.4 Class Structure

  1. 2 hours of instruction
  2. 1 ten minute break
  3. We will ask for feedback to get a feel for understanding

1.5 Learning Recommendations

  1. Use RMD files provided in repo (will get this this later)
  2. Take notes with comments
  3. Execute code along with us
  4. Ask questions in chat

1.6 Endstate

  1. You have git configured on your computer
  2. You have linked git to github or gitlab
  3. You have a repo cloned to your computer.
  4. You can push, pull, and merge git conflicts, on the repository.
  5. You’ve learned a little more about differential equations in R.
